# MdToPdf

Convert markdown file to pdf file using chromium.

## Exemple

```sh
./mdtopdf.py README.pdf
```

## Usage

```sh
mdtopdf --help
```

## Installation

Copy [mdtopdf.py](./mdtopdf.py) where you want to install it.

### Archlinux

Create arch-package and install it.

```sh
makepkg -cfi
```
